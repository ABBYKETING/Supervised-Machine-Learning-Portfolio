# Supervised-Machine-Learning-Portfolio
## Title
- Supervised Machine Learning Portfolio: Classification, Regression, and Time Series Forecasting

## Overview
- This project demonstrates multiple supervised machine learning techniques applied to real-world datasets, including classification, regression, and time series forecasting.

## Projects Included
1. Machine Status Prediction (Classification)
2. Housing Price Prediction (Regression)
3. Energy Consumption Forecasting (Time Series)

## Objectives
1. Apply supervised learning techniques across different problem types
2. Compare multiple machine learning models
3. Perform proper data preprocessing and feature engineering
4. Evaluate models using appropriate metrics
5. Generate insights from model outputs
 
## Methods & Models
- Classification:
    1. Logistic Regression
    2. Random Forest
    3. Decision Tree
    4. KNN
    5. SVC

- Regression:
    1. Linear Regression
    2. Ridge
    3. Lasso
    4. SVR
    5. Decision Tree
    6. Random Forest
    7. Gradient Boosting 

- Time Series:
    1. ARIMA / SARIMA
    2. Exponential Smoothing (ETS)

## Evaluation Metrics
- Classification: Accuracy, Precision, Recall, F1-score
- Regression: RMSE, MAPE
- Time Series: Forecast accuracy + residual analysis

## Key Learnings
- Importance of data preprocessing and feature scaling
- Model selection and comparison
- Handling time-dependent data properly
- Interpreting model results beyond metrics

## Tools & Libraries
1. Python
2. Pandas
3. NumPy
4. Scikit-learn
5. Statsmodels
6. Matplotlib / Seaborn
